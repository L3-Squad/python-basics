# 1, 2 and 3 Bit Basics

## 1 Bit
######
| Decimal | Binary |
|---------|--------|
| 0       | 0      |
| 1       | 1      |

## 2 Bit
####
| Decimal | Binary |
|---------|--------|
| 0       | 00     |
| 1       | 01     |
| 2       | 10     |
| 3       | 11     |

## 3 Bit
####
In the 3-bit table, only decimal numbers up to 7 are included, as the binary representation of 8 (1000) requires 4 bits.

####
| Decimal | Binary |
|---------|--------|
| 0       | 000    |
| 1       | 001    |
| 2       | 010    |
| 3       | 011    |
| 4       | 100    |
| 5       | 101    |
| 6       | 110    |
| 7       | 111    |
